Simple Port Scanner

A lightweight multithreaded Python port scanner that checks a range of ports on a target IP address and logs results to a file.

Features
	•	Scan a custom range of ports
	•	Detect open ports
	•	Fast multithreaded scanning using ThreadPoolExecutor
	•	Saves scan results in a logs/ folder
	•	Displays results in the terminal

Requirements
	•	Python 3.x

No external libraries are required.

How to Run
	1.	Open a terminal in the project folder
  2.	Run:
  python port_scanner.py
  3.	Enter:

	•	Target IP address
	•	Start port
	•	End port
